## Quick Start

### Prerequisites
- Node.js 20+
- npm
- PostgreSQL (for local development)

### Frontend Development
```bash
cd frontend
npm install
npm run dev
```

### Backend Development
```bash
cd backend
npm install
npm run dev
```

## Deployment

- **Frontend:** Vercel (auto-deploy from `main` branch)
- **Backend:** Railway (auto-deploy from `main` branch)
